# ============================================================
#
# Roformer / MDXC separation (BS-Roformer-SW etc.) via run_roformer.py.
#
# audio-separator itself is NOT listed here. It is installed with --no-deps
# (Dockerfile / README step 2) because its metadata says:
#
#     Requires-Dist: diffq (>=0.2)        ; sys_platform != "win32"
#     Requires-Dist: diffq-fixed (>=0.2)  ; sys_platform == "win32"
#
# `diffq` is a C-extension whose newest wheels stop at **cp310**. On any Python
# 3.11+ — including the python:3.11-slim base image — pip finds no wheel, falls
# back to the sdist, and tries to compile it. The image has no gcc (deliberately),
# so `pip install -r requirements.txt` died with:
#
#     error: command 'gcc' failed: No such file or directory
#     ERROR: Could not build wheels for diffq
#
# That is why the GHCR image had not published since the diffq pin appeared: the
# `-r` file is resolved FIRST, so diffq was dragged in and the build failed long
# before the Dockerfile's careful `--no-deps` lines could run. They were dead code.
#
# So audio-separator's real runtime deps are listed here instead, and it is
# installed --no-deps on top. Keep this list in sync when bumping audio-separator
# (`pip show -f audio-separator`, or read its METADATA Requires-Dist).
